Simple business contact information scraper

Instructions:

1. Clone or download project
2. Create an account and get an API key for google maps https://mapsplatform.google.com/
3. Create a file called "config.json" in same directory as place_searching.py
4. Paste following JSON in file replacing 'YOUR_API_KEY' with the API key you received from Google Cloud

{
    "API_KEY": "YOUR_API_KEY"
}
